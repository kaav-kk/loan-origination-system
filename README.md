# Loan Origination System (LOS)

A React.js based Loan Origination System that simulates the loan application workflow used in banking.

## Features

- Role-based Login
- Create Loan
- Multi-step Loan Application
- Loan Dashboard
- Approval Pending Queue
- Loan Sanctioned Queue
- View Loan Details
- Local Storage Persistence

## Technologies Used

- React.js
- JavaScript
- HTML
- CSS
- React Router
